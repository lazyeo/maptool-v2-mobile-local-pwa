TECHNICAL SUPPORT & TROUBLESHOOTING

Issue: Data appears incomplete or outdated

  Command: node process-rolleston-addresses.js

Issue: Cannot parse JSON file
  Troubleshooting:
    1. Verify file not corrupted: wc -l rolleston-addresses.json
    2. Validate JSON: jq empty rolleston-addresses.json
    3. Check encoding: file -I rolleston-addresses.json
    4. Re-download if corrupted

Issue: Coordinates not displaying on map
  Troubleshooting:
    1. Verify lat/lon values are in WGS84 (-180 to 180 for lon, -90 to 90 for lat)
    2. Check: -43.5913 and 172.3721 are valid New Zealand coordinates
    3. Ensure map library supports WGS84 (most do)
    4. Debug: log first address coordinates

Issue: Missing addresses for known locations
  Note: OSM data is community-contributed and completeness varies
  Solution:
    1. Verify address is tagged in OSM (check osm.org)
    2. Add missing address to OpenStreetMap
    3. Wait 2-3 days for Overpass API cache to update
    4. Re-run extraction script
